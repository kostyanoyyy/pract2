@' консольная утилита (C, getopt_long, CMake)

Функционал
Утилита выполняет несколько действий по аргументам командной строки:

-u, --users — вывести список пользователей и их домашние директории (из /etc/passwd), отсортировано по имени.
-p, --processes — вывести список запущенных процессов (из /proc), отсортировано по PID. Печатается шапка: PID COMMAND.
-h, --help — показать справку и завершить работу.
-l PATH, --log PATH — перенаправить стандартный вывод (stdout) в файл PATH.
-e PATH, --errors PATH — перенаправить поток ошибок (stderr) в файл PATH.


Ввод/вывод и коды возврата
Ввод: только аргументы командной строки.
Вывод: информация в stdout; сообщения об ошибках — в stderr (или в файл через -e).
Коды возврата: 0 — успех; 1/2 — ошибки открытия пути/записи и т.п. (см. сообщения).

cmake --build .build
